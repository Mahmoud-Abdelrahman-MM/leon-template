# Leon

**Leon** is a modern, fully responsive agency website template built with HTML and CSS. It is ideal for creative agencies, freelancers, and startups seeking a clean and professional online presence.

## Features

- Responsive layout for mobile, tablet, and desktop
- Clean, minimal design
- Modern UI/UX
- Easy to customize
- Organized project structure

## Technologies

- HTML5
- CSS3

## Live Demo

- https://mahmoud-abdelrahman-mm.github.io/leon-template/

## Project Structure

```
index.html
css/
  ├── all.min.css
  ├── normalize.css
  └── style.css
images/
webfonts/
README.md
```

## Usage

1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Customize content and styles as needed.

## Purpose

This project was created as a portfolio piece and a learning exercise to demonstrate responsive HTML/CSS and clean agency-style design.
